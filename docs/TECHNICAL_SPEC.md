# Техническое Задание: Улучшение UI/UX CogniFlex

**Версия:** 1.0  
**Дата:** 17 марта 2026  
**Статус:** Черновик  

---

## 1. Введение

### 1.1 Назначение Документа

Этот документ описывает техническое задание на улучшение пользовательского интерфейса и пользовательского опыта (UI/UX) системы CogniFlex.

### 1.2 Scope (Границы Проекта)

**Включено:**
- Оптимизация верхней панели (убрать пустое место)
- Исправление навигации (кнопки → визуальные вкладки)
- Улучшение статус-бара
- Документация и исследования

**Не включено:**
- Миграция на CustomTkinter (отдельный проект)
- Мобильная адаптация
- Облачная синхронизация

## 3. Нефункциональные Требования

### 3.1 Производительность

- Время запуска GUI: < 30 секунд
- Время переключения между модулями: < 500ms
- Использование памяти: без увеличения

### 3.2 Совместимость

- Python 3.10+
- Tkinter (встроенный)
- Windows 10/11

### 3.3 Тестирование

- Ручное тестирование каждого изменения
- Проверка работоспособности всех 8 модулей
- Тест на разных разрешениях экрана


## 5. Техническая Реализация

### 5.1 Изменяемые Файлы

| Файл | Изменения |
|------|-----------|
| `cogniflex/gui/core_gui.py` | Методы `_create_navbar()`, `_create_status_bar()`, `_create_interface()` |
| `cogniflex/gui/core_gui.py` | Добавить метод `_update_nav_visual_state()` |
| `cogniflex/gui/core_gui.py` | Метод `_switch_view()` - добавить визуальное обновление |

### 5.2 Алгоритм Реализации FR-2.1 (Подсветка навигации)

```python
# 1. Сохранить ссылки на кнопки навигации
self._nav_buttons = {}
for text, view_id in nav_items:
    btn = ttk.Button(...)
    self._nav_buttons[view_id] = btn

# 2. Обновить при переключении
def _switch_view(self, view_id: str):
    # ... существующий код ...
    
    # Обновить подсветку
    self._update_nav_visual_state(view_id)

# 3. Метод подсветки
def _update_nav_visual_state(self, active_view_id: str):
    for view_id, btn in self._nav_buttons.items():
        if view_id == active_view_id:
            btn.configure(style="NavActive.TButton")
        else:
            btn.configure(style="Nav.TButton")
```

## 7. Критерии Приёмки

### 7.1 Приёмка FR-1

- [ ] Нет видимого разрыва между навигацией и контентом
- [ ] Общая высота верхней части < 50px
- [ ] Интерфейс корректно отображается при 1024x768

### 7.2 Приёмка FR-2

- [ ] Пользователь видит активный раздел (цветовая подсветка)
- [ ] Все 8 модулей переключаются корректно
- [ ] Нет визуальных артефактов при переключении

### 7.3 Приёмка FR-3

- [ ] Все метрики отображаются корректно
- [ ] Термины понятны нетехническому пользователю
- [ ] Status bar не перекрывается контентом
