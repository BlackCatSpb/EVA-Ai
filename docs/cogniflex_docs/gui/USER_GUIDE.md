# Руководство пользователя GUI CogniFlex

Документ представляет собой руководство по использованию графического интерфейса системы CogniFlex. Здесь описаны установка, запуск, основные функции и навигация по модулям.

## Установка и запуск

### Системные требования

Для работы GUI CogniFlex требуется Python версии 3.8 или выше. Также необходимо установить следующие зависимости: Tkinter для графического интерфейса, PyTorch для работы моделей машинного обучения, а также дополнительные пакеты, указанные в файле requirements.txt.

### Установка зависимостей

Установите все необходимые зависимости с помощью менеджера пакетов pip:

pip install -r requirements.txt

Убедитесь, что все зависимости установлены корректно, прежде чем запускать приложение.

### Запуск приложения

Самый простой способ запуска - использование скрипта запуска. В корневой директории проекта найдите файл run.py или start.bat и выполните его. На Windows это можно сделать двойным щелчком по файлу start.bat.

Альтернативно, вы можете запустить приложение из командной строки:

python run.py

При успешном запуске откроется главное окно приложения с интерфейсом чата.

### Проверка готовности системы

После запуска дождитесь загрузки моделей машинного обучения. В статус-баре отображается индикатор готовности. Когда система будет готова к работе, индикатор изменит цвет на зеленый.

## Основные функции

### Чат с системой

Основной интерфейс взаимодействия с CogniFlex - текстовый чат. Введите ваш вопрос или запрос в поле ввода в нижней части окна и нажмите Enter для отправки. Система обработает запрос и отправит ответ.

Вы можете использовать форматирование Markdown в ваших сообщениях. Поддерживается жирный текст, курсив, код и ссылки. Система автоматически распознает гиперссылки и делает их кликабельными.

### Панель рассуждений

Под основной областью чата расположена панель рассуждений. Она показывает внутренний процесс мышления системы при обработке запроса. Панель можно развернуть или свернуть с помощью кнопки в ее заголовке.

### История сообщений

Все сообщения сохраняются в историю чата. Вы можете просматривать предыдущие диалоги с помощью стрелок Вверх и Вниз на клавиатуре в поле ввода. История автоматически загружается при следующем запуске приложения.

### Импорт документов

Вы можете обучить систему на собственных документах. Нажмите кнопку Импорт в области ввода сообщения и выберите файл. Поддерживаются форматы TXT, MD, LOG, PDF и EPUB. После импорта система автоматически начнет обучение на новом материале.

### Автодиалог

Кнопка Автодиалог запускает режим самостоятельного общения системы. В этом режиме CogniFlex генерирует вопросы к самой себе и анализирует собственные ответы. Это полезно для непрерывного обучения и выявления противоречий в знаниях.

## Навигация по модулям

### Панель навигации

В верхней части окна расположена панель навигации с кнопками для переключения между модулями. Доступны следующие модули:

Модуль Чат предоставляет интерфейс для диалога с системой. Модуль Аналитика отображает статистику использования и производительность системы. Модуль Граф знаний визуализирует структуру накопленных знаний и связи между концепциями. Модуль Противоречия показывает обнаруженные логические конфликты в базе знаний. Модуль Память отображает состояние системы кэширования и использование ресурсов. Модуль Обучение управляет процессом обучения модели. Модуль Нейроморфика визуализирует работу нейроморфных компонентов. Модуль Настройки позволяет изменить параметры системы.

### Переключение между модулями

Для переключения нажмите соответствующую кнопку на панели навигации. Текущий модуль подсвечивается. Интерфейс переключается мгновенно без перезагрузки приложения.

### Статус-бар

В нижней части окна расположен статус-бар с важной информацией. Здесь отображаются: индикатор соединения с системой, текущее время, загрузка процессора, использование памяти, количество противоречий, показатель попаданий в кэш и использование кэша.

## Настройки системы

### Смена темы

Вы можете переключиться между светлой и темной темой оформления. Откройте модуль Настройки и найдите параметр Тема. Выбор применяется немедленно без перезагрузки.

### Параметры уведомлений

В модуле Настройки можно настроить уведомления. Укажите длительность отображения и разрешите или запретите определенные типы уведомлений.

### Кэширование

Параметры кэширования находятся в модуле Память. Здесь можно очистить кэш, просмотреть статистику использования и настроить лимиты памяти.

## Часто задаваемые вопросы

### Приложение не запускается

Убедитесь, что все зависимости установлены. Проверьте версию Python - требуется 3.8 или выше. Посмотрите сообщения об ошибках в консоли. Частая причина - отсутствие необходимых пакетов.

### Система медленно отвечает

Проверьте загрузку системы в статус-баре. Если процессор перегружен, закройте другие приложения. Также убедитесь, что достаточно оперативной памяти.

### Как очистить историю чата

Откройте контекстное меню в области чата (правая кнопка мыши) и выберите Очистить чат. Подтвердите действие в появившемся диалоге.

### Как изменить размер окна

Размер окна можно изменить стандартными средствами - перетаскиванием границ окна. Приложение запоминает размер при следующем запуске.

### Где хранятся настройки

Настройки и кэш хранятся в директории cogniflex_gui_cache в папке модуля GUI. Там же находится история чата и другие пользовательские данные.

### Что делать при обнаружении противоречий

Откройте модуль Противоречия для детального просмотра. Система автоматически уведомляет о новых противоречиях. Вы можете проанализировать каждое противоречие и принять решение о его разрешении.

### Как экспортировать диалог

Используйте функцию Копировать все из контекстного меню. Весь текст будет скопирован в буфер обмена, откуда его можно вставить в любой текстовый редактор.

### Поддерживаются ли горячие клавиши

Да, поддерживаются следующие горячие клавиши: Enter отправляет сообщение, Shift+Enter создает новую строку, Ctrl+C копирует выделенный текст, Ctrl+V вставляет текст, Ctrl+A выделяет весь текст, F1 показывает справку.

## Устранение неполадок

### Ошибка загрузки модели

Если при запуске возникает ошибка загрузки модели, проверьте наличие файлов моделей в директории models. При необходимости повторно загрузите модели. Также убедитесь, что достаточно оперативной памяти для загрузки модели.

### Зависание интерфейса

При зависании интерфейса попробуйте выполнить горячую перезагрузку. Нажмите кнопку Горячая перезагрузка на панели навигации. Это перезагрузит модули без полной остановки системы.

### Потеря данных

Не беспокойтесь - история чата и настройки сохраняются автоматически. При следующем запуске все данные будут восстановлены. Если данные все же пропали, проверьте права доступа к директории кэша.
