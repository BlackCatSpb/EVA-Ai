# FractalQwenGenerator: Полное описание единой архитектуры и уникальных методов

## 1. Общая концепция системы

**FractalQwenGenerator** — это интегрированная система генерации текста на основе модели **Ruadapt Qwen3 4B** (OpenVINO GenAI) и **фрактального графа знаний FractalGraphV2**. Система реализует архитектуру «Слоёный пирог» (L0–L3), в которой каждый слой отвечает за определённый аспект обработки, а все компоненты функционируют как единое целое.

Ключевые возможности системы:

- Адаптивная генерация ответов на короткие запросы с автоматической настройкой параметров под предметную область.
- Превращение больших документов во **временное расширение графа** для контекстно‑зависимой работы.
- Батчевая обработка массивных текстов с сохранением иерархической структуры через структурные токены.
- Непрерывное самообучение за счёт сбора статистики (L1) и дообучения LoRA‑адаптеров (L0) на основе знаний графа (L3).
- Гибридная токенизация с виртуальными токенами, соответствующими узлам графа.

## 2. Слоистая архитектура (L0–L3)

| Слой | Название | Компонент | Функции |
|------|----------|-----------|---------|
| **L0** | Инференс | OpenVINO GenAI + LoRAManager | Генерация текста, динамическое применение доменных адаптеров |
| **L1** | Профили активаций | `ActivationProfiler` | Сбор статистики (латентность, энтропия, качество), построение центроидов доменов |
| **L2** | Маршрутизация | `RoutingEngine` | Выбор параметров генерации (температура, top_p, max_tokens) на основе домена запроса и профилей L1 |
| **L3** | Когнитивная память | `FractalGraphV2` (постоянный + временные подграфы) | Хранение концептов, фактов, правил маршрутизации, профилей активаций, а также временное представление загружаемых документов |

## 3. Структура единого файла .fmf (Fractal Model Format)

Для переносимости системы разработан формат `.fmf`, объединяющий модель, граф и метаданные в одном файле:

```
┌─────────────────────────────────────────────────────────────────────┐
│ Magic: "FMF\0" (4 bytes) + Version (4 bytes)                        │
├─────────────────────────────────────────────────────────────────────┤
│ Section Index (JSON) – смещения и размеры всех секций                │
├─────────────────────────────────────────────────────────────────────┤
│ Tensor Data (GGUF‑совместимый блок)                                  │
│   - Метаданные модели                                                │
│   - Тензоры Condensed + Extended с префиксами имён                   │
├─────────────────────────────────────────────────────────────────────┤
│ Fractal Graph Section                                                │
│   - SQLite база графа (сжатая zstd)                                  │
│   - Индекс эмбеддингов узлов                                         │
├─────────────────────────────────────────────────────────────────────┤
│ Tokenizer Mapping Section                                            │
│   - Таблица виртуальных токенов: ID → (слой, node_id, тип, эмбеддинг)│
│   - Сериализованный Aho‑Corasick автомат                             │
├─────────────────────────────────────────────────────────────────────┤
│ Routing & Profiles (L1/L2 Cache)                                     │
│   - Сериализованные routing_rule узлы                                 │
│   - Сериализованные activation_profile узлы                           │
└─────────────────────────────────────────────────────────────────────┘
```

### 4.4. EmbeddingProjector

Преобразует эмбеддинги из размерности энкодера (например, 1024) в размерность скрытого состояния модели Qwen3 (3584).

```python
class EmbeddingProjector:
    def __init__(self, src_dim: int, tgt_dim: int, method: str = "repeat")
    def __call__(self, x: np.ndarray) -> np.ndarray
```

**Методы проекции:**
- `"repeat"` – повторение вектора с сохранением нормы (быстро, без обучения).
- `"linear"` – через обученный линейный слой (выше качество).

### 4.8. DocumentKnowledgeLoader

Превращает документ во временный подграф.

```python
class DocumentKnowledgeLoader:
    def __init__(self, graph, encoder, tokenizer, hybrid_tokenizer, projection)
    def load_document(self, doc_text: str, doc_id: str) -> str  # возвращает session_id
```

**Алгоритм:**
1. Разбивает текст на параграфы.
2. Для каждого параграфа вычисляет эмбеддинг и создаёт временный узел типа `temp_paragraph`.
3. Связывает последовательные параграфы рёбрами `next`.
4. Добавляет временные узлы в гибридный токенизатор.

### 4.9. BatchPipeline (Chunker, Scheduler, Aggregator)

Компоненты для обработки больших документов с сохранением структуры.

| Компонент | Метод | Описание |
|-----------|-------|----------|
| `FractalChunker` | `chunk_document(text, doc_id)` | Разбивает текст на чанки с добавлением структурных токенов (`<|doc_start|>`, `<|seq N|>` и т.д.) |
| `BatchScheduler` | `process_chunks(chunks, task)` | Обрабатывает чанки батчами, используя `ThreadPoolExecutor` |
| `StructuralAggregator` | `aggregate(results, aggregation_type)` | Объединяет результаты чанков в итоговый ответ (map‑reduce или иерархически) |

| `<|header_level1 {title}|>` | 200010 | Заголовок первого уровня |
| ... | ... | ... |

## 6. Жизненный цикл обработки запроса

### 6.1. Короткий диалоговый запрос

1. Гибридный токенизатор находит в запросе узлы L3 (постоянные + временные) и заменяет их на виртуальные токены.
2. Определяется домен запроса (по виртуальным токенам или эмбеддингу).
3. RoutingEngine (L2) возвращает `GenerationConfig` с оптимальными параметрами.
4. Если доступен LoRA‑адаптер для домена, он применяется к модели.
5. Строится обогащённый промпт (добавляются факты из графа).
6. OpenVINO Pipeline генерирует ответ.
7. ActivationProfiler (L1) асинхронно обновляет профиль домена.

### 6.2. Загрузка документа для последующих вопросов

1. `DocumentKnowledgeLoader` создаёт временный подграф документа.
2. Гибридный токенизатор расширяет Aho‑Corasick автомат узлами документа.
3. На любой вопрос по документу система автоматически использует соответствующие виртуальные токены.

### 6.3. Батчевая обработка огромной книги

1. `FractalChunker` разбивает текст на чанки с структурными токенами.
2. `BatchScheduler` группирует чанки в батчи и параллельно генерирует промежуточные ответы.
3. `StructuralAggregator` объединяет результаты в итоговый ответ, учитывая структурные токены.

## 7. Преимущества единой архитектуры

- **Гибкость:** одинаково эффективна для коротких запросов, диалогов с документами и пакетной обработки.
- **Контекстная связность:** временные подграфы и структурные токены гарантируют, что модель «видит» документ целиком.
- **Самообучение:** LoRA‑адаптеры и профили L1 позволяют системе адаптироваться к предметным областям.
- **Производительность:** батчевая обработка и Aho‑Corasick токенизация минимизируют накладные расходы.
- **Переносимость:** единый формат `.fmf` упрощает развёртывание и обмен моделями с встроенными знаниями.

## 1. FractalQwenGenerator

Главный класс, инкапсулирующий модель, граф, токенизатор и сервисы L1/L2.

### 1.1. `__init__`

Инициализирует все компоненты системы.

```python
def __init__(
    self,
    model_path: str,
    graph_path: str,
    device: str = "CPU",
    encoder_model: str = "intfloat/multilingual-e5-large",
    lora_adapters_dir: str = "./lora_adapters"
):
    """
    Параметры:
        model_path: путь к директории с OpenVINO IR моделью (.xml, .bin)
        graph_path: путь к SQLite файлу фрактального графа
        device: устройство инференса ('CPU', 'GPU', 'NPU')
        encoder_model: имя модели SentenceTransformer для эмбеддингов
        lora_adapters_dir: директория с предобученными LoRA адаптерами
    """
    self.device = device
    self.model_path = Path(model_path)
    self.graph_path = Path(graph_path)
    
    # Загрузка OpenVINO пайплайна
    import openvino_genai as ov_genai
    self.pipe = ov_genai.LLMPipeline(str(model_path), device)
    
    # Токенизатор Qwen3
    from transformers import AutoTokenizer
    self.tokenizer = AutoTokenizer.from_pretrained(str(model_path), trust_remote_code=True)
    
    # Энкодер эмбеддингов
    from sentence_transformers import SentenceTransformer
    self.encoder = SentenceTransformer(encoder_model)  # 1024-dim
    
    # Проектор эмбеддингов 1024 -> 3584
    from .embedding_projection import EmbeddingProjector
    self.projection = EmbeddingProjector(
        src_dim=1024,
        tgt_dim=self.QWEN3_4B_HIDDEN_SIZE,  # 3584
        method='repeat'
    )
    
    # Фрактальный граф
    from eva_ai.memory.fractal_graph_v2 import FractalGraphV2
    self.graph = FractalGraphV2(str(graph_path))
    self.graph.register_node_types()
    
    # Гибридный токенизатор
    from .hybrid_tokenizer import HybridTokenizer
    self.hybrid_tokenizer = HybridTokenizer(
        graph=self.graph,
        base_tokenizer=self.tokenizer,
        encoder=self.encoder,
        projection=self.projection,
        vocab_offset=self.QWEN3_4B_VOCAB_SIZE
    )
    
    # L1 и L2 сервисы
    from .pie_services import ActivationProfiler, RoutingEngine
    self.profiler = ActivationProfiler(self.graph, self.encoder, self.projection)
    self.router = RoutingEngine(self.graph, self.encoder)
    
    # LoRA менеджер
    from .lora_manager import LoRAManager
    self.lora_manager = LoRAManager(self.pipe, lora_adapters_dir)
```

### 1.2. `generate`

Основной метод диалоговой генерации.

```python
def generate(
    self,
    prompt: str,
    mode: str = "condensed",
    use_lora: bool = True,
    use_embedding_prefix: bool = False,
    enable_thinking: bool = False,
    context_nodes: Optional[List[str]] = None
) -> Dict[str, Any]:
    """
    Параметры:
        prompt: пользовательский запрос
        mode: 'condensed' (быстрый) или 'extended' (развёрнутый)
        use_lora: применять LoRA адаптер для домена
        use_embedding_prefix: вставлять эмбеддинги узлов как префикс
        enable_thinking: включить цепочку рассуждений Qwen3
        context_nodes: список ID узлов графа для принудительного включения в контекст
    
    Возвращает:
        Словарь с полями:
            response: str
            domain: str
            config: dict (параметры генерации)
            latency_ms: float
            virtual_tokens_used: int
            lora_applied: Optional[str]
    """
    import time
    import threading
    
    # 1. Гибридная токенизация + извлечение эмбеддингов
    tokens, prefix_embs = self.hybrid_tokenizer.encode(
        prompt, layers={2, 3}, return_embeddings=use_embedding_prefix
    )
    
    # 2. Определение домена
    domain = self._extract_domain_from_tokens(tokens)
    
    # 3. Получение параметров из L2
    query_emb = self.projection(self.encoder.encode(prompt))
    config = self.router.get_generation_config(domain_hint=domain, query_embedding=query_emb)
    
    # 4. Применение LoRA адаптера
    lora_applied = None
    if use_lora and domain != "general":
        lora_applied = self.lora_manager.load_adapter(domain)
        if lora_applied:
            self.lora_manager.apply_adapter(lora_applied, alpha=0.8)
    
    # 5. Построение обогащённого промпта
    enriched_prompt = self._build_context_prompt(prompt, domain, context_nodes)
    messages = [{"role": "user", "content": enriched_prompt}]
    full_prompt = self.tokenizer.apply_chat_template(
        messages, tokenize=False, add_generation_prompt=True,
        enable_thinking=enable_thinking
    )
    
    # 6. Генерация
    start_time = time.time()
    if use_embedding_prefix and prefix_embs is not None:
        response = self._generate_with_prefix(full_prompt, prefix_embs, config)
    else:
        response = self.pipe.generate(
            full_prompt,
            max_new_tokens=config.max_new_tokens,
            temperature=config.temperature,
            top_p=config.top_p,
            repetition_penalty=config.repetition_penalty,
            do_sample=config.temperature > 0.0
        )
    latency_ms = (time.time() - start_time) * 1000
    
    # 7. Асинхронное обновление профиля L1
    def update_profile():
        fingerprint = self.profiler.compute_fingerprint(prompt, response)
        self.profiler.update(
            domain=domain, model_id="qwen3-4b",
            fingerprint=fingerprint, latency_ms=latency_ms
        )
        if config.rule_id:
            self.router.update_rule_stats(config.rule_id, success=None)
    
    threading.Thread(target=update_profile, daemon=True).start()
    
    return {
        "response": response,
        "domain": domain,
        "config": config.to_dict(),
        "latency_ms": latency_ms,
        "virtual_tokens_used": self.hybrid_tokenizer.count_virtual_tokens(tokens),
        "lora_applied": lora_applied
    }
```

### 1.3. `load_document`

Создаёт временный подграф документа для контекстной работы.

```python
def load_document(self, doc_text: str, doc_id: str) -> str:
    """
    Параметры:
        doc_text: полный текст документа
        doc_id: уникальный идентификатор документа
    
    Возвращает:
        session_id: идентификатор временной сессии
    """
    from .document_loader import DocumentKnowledgeLoader
    loader = DocumentKnowledgeLoader(
        graph=self.graph,
        encoder=self.encoder,
        tokenizer=self.tokenizer,
        hybrid_tokenizer=self.hybrid_tokenizer,
        projection=self.projection
    )
    return loader.load_document(doc_text, doc_id)
```

### 1.4. `unload_document`

Удаляет временный подграф документа.

```python
def unload_document(self, session_id: str):
    """
    Параметры:
        session_id: идентификатор сессии, полученный из load_document()
    """
    self.hybrid_tokenizer.remove_temporary_nodes(session_id)
    self.graph.rollback_temp_session(session_id)
```

```python
def process_large_document(
    self,

    doc_id: str,

        task: тип обработки ('summarize', 'qa', 'extract_entities')

    Возвращает:
        Словарь с полями:
            summary: str (итоговый результат)
            num_chunks: int
            processing_time: float
            structural_metadata: dict
    """
    from .batch_pipeline import FractalChunker, BatchScheduler, StructuralAggregator
    
    # 1. Чанкинг с добавлением структурных токенов
    chunker = FractalChunker(self.graph, self.tokenizer, max_chunk_tokens)
    chunks = chunker.chunk_document(document_text, doc_id)
    
    # 2. Параллельная обработка батчами
    scheduler = BatchScheduler(self, max_batch_size=batch_size)
    chunk_results = scheduler.process_chunks(chunks, task)
    
    # 3. Агрегация с учётом структуры
    aggregator = StructuralAggregator(self.graph)
    final_output = aggregator.aggregate(chunk_results)
    
    return {
        "summary": final_output,
        "num_chunks": len(chunks),
        "processing_time": sum(r.latency_ms for r in chunk_results) / 1000,
        "structural_metadata": self._extract_structure(chunks)
    }
```

### 2.2. `encode`

```python
def encode(
    self,
    text: str,
    layers: Set[int] = {2, 3},
    return_embeddings: bool = False
) -> Tuple[List[int], Optional[np.ndarray]]:
    """
    Токенизирует текст, заменяя найденные узлы графа на виртуальные токены.
    
    Параметры:
        text: входной текст
        layers: какие слои включать (2 - правила, 3 - концепты/факты)
        return_embeddings: возвращать ли массив эмбеддингов найденных узлов
    
    Возвращает:
        (list токенов, опционально np.ndarray эмбеддингов)
    """
    tokens = []
    prefix_embs = []
    pos = 0
    text_lower = text.lower()
    
    for end_idx, (token_id, matched_text) in self.automaton.iter(text_lower):
        start_idx = end_idx - len(matched_text) + 1
        # BPE-токенизация текста до совпадения
        if pos < start_idx:
            prefix = text[pos:start_idx]
            tokens.extend(self.base_tokenizer.encode(prefix))
        # Виртуальный токен, если слой разрешён
        vt = self.virtual_tokens.get(token_id)
        if vt and vt["layer"] in layers:
            tokens.append(token_id)
            if return_embeddings and vt.get("embedding") is not None:
                prefix_embs.append(vt["embedding"])
        pos = end_idx + 1
    
    # Остаток текста
    if pos < len(text):
        tokens.extend(self.base_tokenizer.encode(text[pos:]))
    
    emb_array = np.stack(prefix_embs) if prefix_embs else None
    return tokens, emb_array
```

```python
def add_temporary_nodes(self, session_id: str, graph: FractalGraphV2):
    """Добавляет временные узлы указанной сессии в автомат."""
    for node_id in graph.temp_sessions.get(session_id, []):
        node = graph.get_node(node_id)
        if node:
            text = node.content.lower()
            token_id = self.VIRTUAL_TOKEN_OFFSET + len(self.virtual_tokens)
            # Получение эмбеддинга узла
            emb = node.embedding
            if emb is None:
                emb = self.encoder.encode(node.content)
                emb = self.projection(emb)
            else:
                emb = np.frombuffer(emb, dtype=np.float32)
            
            self.virtual_tokens[token_id] = {
                "node_id": node_id,
                "layer": 3,
                "node_type": node.node_type,
                "text": node.content,
                "embedding": emb,
                "temporary": True,
                "session_id": session_id
            }
            self.automaton.add_word(text, (token_id, text))
    self.automaton.make_automaton()
```

```python
def create_activation_profile(
    self,
    domain: str,
    model_id: str,
    quant_profile: str,
    initial_embedding: np.ndarray = None
) -> str:
    """Создаёт узел activation_profile."""

    metadata = {
        "domain": domain,
        "model_id": model_id,
        "quant_profile": quant_profile,
        "avg_entropy": 0.0,
        "avg_latency_ms": 0.0,
        "avg_quality": 0.0,
        "sample_count": 0,
        "last_accessed": time.time()
    }
    emb = np.zeros(768, dtype=np.float32) if initial_embedding is None else initial_embedding
    return self.add_node(
        content=content,
        node_type="activation_profile",
        level=2,
        embedding=emb.tobytes(),
        metadata=metadata,
        temporal_weight=1.0,
        domain_lambda=0.05
    )

def update_activation_profile(
    self,
    node_id: str,
    new_fingerprint: np.ndarray,
    entropy: float,
    latency_ms: float,
    quality: float = 0.0
):
    """Инкрементально обновляет центроид профиля."""
    node = self.get_node(node_id)
    metadata = node.metadata.copy()
    n = metadata.get("sample_count", 0)
    old_centroid = np.frombuffer(node.embedding, dtype=np.float32)
    new_centroid = (old_centroid * n + new_fingerprint) / (n + 1)
    
    metadata["avg_entropy"] = (metadata["avg_entropy"] * n + entropy) / (n + 1)
    metadata["avg_latency_ms"] = (metadata["avg_latency_ms"] * n + latency_ms) / (n + 1)
    metadata["avg_quality"] = (metadata["avg_quality"] * n + quality) / (n + 1)
    metadata["sample_count"] = n + 1
    metadata["last_accessed"] = time.time()
    
    self.update_node(
        node_id=node_id,
        embedding=new_centroid.tobytes(),
        metadata=metadata,
        temporal_weight=min(1.0, node.temporal_weight + 0.01)
    )
```

```python
def get_routing_rule(
    self,
    domain_hint: str = None,
    query_embedding: np.ndarray = None
) -> Optional[Dict]:
    """Возвращает наиболее подходящее правило."""
    if domain_hint:
        concepts = self.search_nodes(content=domain_hint, node_type="concept")
        if concepts:
            rules = self.get_related_nodes(concepts[0]["id"], "applies_to_domain", "routing_rule")
            if rules:
                return max(rules, key=lambda r: r["metadata"].get("priority", 0))
    
    if query_embedding is not None:
        profiles = self.semantic_search(query_embedding, top_k=3, node_type="activation_profile")
        for prof in profiles:
            domain = prof["metadata"]["domain"]
            return self.get_routing_rule(domain_hint=domain)
    return None
```

## 4. EmbeddingProjector

```python
class EmbeddingProjector:
    def __init__(self, src_dim: int, tgt_dim: int, method: str = "repeat"):
        self.src_dim = src_dim
        self.tgt_dim = tgt_dim
        self.method = method
        if method == "linear":
            import torch.nn as nn
            self.linear = nn.Linear(src_dim, tgt_dim, bias=False)
            nn.init.orthogonal_(self.linear.weight)
    
    def __call__(self, x: np.ndarray) -> np.ndarray:
        if x.shape[-1] == self.tgt_dim:
            return x
        if self.method == "repeat":
            repeats = (self.tgt_dim + self.src_dim - 1) // self.src_dim
            x_tiled = np.tile(x, repeats)[:self.tgt_dim]
            norm_orig = np.linalg.norm(x)
            norm_tiled = np.linalg.norm(x_tiled)
            if norm_tiled > 0:
                x_tiled = x_tiled * (norm_orig / norm_tiled)
            return x_tiled.astype(np.float32)
        elif self.method == "linear":
            import torch
            with torch.no_grad():
                x_t = torch.from_numpy(x).float()
                return self.linear(x_t).numpy()
        else:
            raise ValueError(f"Unknown method: {self.method}")
```

## 9. BatchPipeline (Chunker, Scheduler, Aggregator)

### 9.1. FractalChunker

```python
class FractalChunker:
    def __init__(self, graph, tokenizer, max_chunk_tokens: int = 2048):
        self.graph = graph
        self.tokenizer = tokenizer
        self.max_chunk_tokens = max_chunk_tokens
    
    def chunk_document(self, text: str, doc_id: str) -> List[Chunk]:
        paragraphs = text.split('\n\n')
        chunks = []
        current_chunk = []
        current_tokens = 0
        seq = 0
        
        for para in paragraphs:
            para_tokens = len(self.tokenizer.encode(para))
            if current_tokens + para_tokens > self.max_chunk_tokens and current_chunk:
                chunks.append(Chunk(
                    id=f"{doc_id}_{seq}",
                    doc_id=doc_id,
                    seq=seq,
                    content='\n\n'.join(current_chunk),
                    prompt=f"<|doc_start id={doc_id}|><|seq {seq}|>\n{'\n\n'.join(current_chunk)}\n<|doc_end|>"
                ))
                seq += 1
                current_chunk = [para]
                current_tokens = para_tokens
            else:
                current_chunk.append(para)
                current_tokens += para_tokens
        
        if current_chunk:
            chunks.append(Chunk(
                id=f"{doc_id}_{seq}",
                doc_id=doc_id,
                seq=seq,
                content='\n\n'.join(current_chunk),
                prompt=f"<|doc_start id={doc_id}|><|seq {seq}|>\n{'\n\n'.join(current_chunk)}\n<|doc_end|>"
            ))
        
        return chunks
```

### 9.2. BatchScheduler

```python
class BatchScheduler:
    def __init__(self, generator: FractalQwenGenerator, max_batch_size: int = 8):
        self.generator = generator
        self.max_batch_size = max_batch_size
    
    def process_chunks(self, chunks: List[Chunk], task: str) -> List[ChunkResult]:
        from concurrent.futures import ThreadPoolExecutor
        results = []
        
        chunks_sorted = sorted(chunks, key=lambda c: len(c.content))
        
        for i in range(0, len(chunks_sorted), self.max_batch_size):
            batch = chunks_sorted[i:i+self.max_batch_size]
            prompts = [c.prompt for c in batch]
            
            with ThreadPoolExecutor(max_workers=len(batch)) as executor:
                futures = [executor.submit(self.generator.pipe.generate, p) for p in prompts]
                outputs = [f.result() for f in futures]
            
            for chunk, output in zip(batch, outputs):
                results.append(ChunkResult(chunk_id=chunk.id, output=output))
        
        # Восстановить порядок
        results.sort(key=lambda r: r.chunk_id)
        return results
```
